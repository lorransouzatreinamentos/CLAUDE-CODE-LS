# CLAUDE CODE LS

Backup e sincronização das configurações globais do Claude Code.

Contém todos os agentes, skills e regras do framework **Segunda-feira** instalados localmente.

## Estrutura

```
commands/      # 31 agentes (slash commands)
skills/        # 8 skills de negócio e marketing
agents/        # 7 sub-agentes especializados
rules/         # Regras de workflow e autoridade
organization/  # Estrutura de departamentos
CLAUDE.md      # Configuração global
```

## Como usar no Claude.ai (web/mobile)

1. Conecte este repositório ao Claude.ai
2. Use os arquivos de `commands/` e `skills/` como instruções de Projects
